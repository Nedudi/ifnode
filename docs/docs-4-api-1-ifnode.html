[[Docs|docs]] → [[API|docs/api]] → ifnode

Description:

Returns instance of ifnode by options:

[class Application] ifnode([Object options])

Parameters:

Option Type Description Optional Default
alias String "Nice" name of application true Equal to id
project_folder String Full path to application root project folder true Equal to process.argv[1]
environment String Name of application config (folder ${project_folder}/config) true -
projectFolder Alias of option .project_folder
env Alias of option .environment

Example:

var app = ifnode({
    project_folder: __dirname,
    alias: 'application',
    env: 'dev'
});
